# Campus Pulse – Smart Campus Companion

Campus Pulse is an Android application built to act as a smart companion for students by centralizing essential academic, navigation, and campus services into one intuitive mobile app.

## App Pages & Functionality

### Home (Dashboard)
Provides an overview of the student’s academic status, including attendance percentage, CGPA, number of classes for the day, and unread notices. Acts as the primary entry point for quick access to key features.

### Academics
Allows students to explore academic-related services such as attendance tracking, timetable access, notices, marks & results, classroom availability, and campus services.

### Timetable
Displays the daily class schedule with real-time status indicators such as ongoing sessions, upcoming classes, breaks, and completed lectures for better time management.

### Campus Map
An interactive campus navigation screen that visually represents important buildings and facilities (academic blocks, library, canteen, hostels, parking, etc.) using icons and connections instead of static images.

### Emergency
Provides quick access to emergency support options within the campus environment, designed for safety and immediate assistance.

### Profile
Displays student profile information and academic details such as branch, semester, section, and access to app-related settings.

## Tech Stack
- Android (Jetpack Compose)
- Kotlin
- Material 3 Design
- Android Studio

## Objective
To demonstrate a practical **Smart Campus Solution** that improves accessibility, awareness, and navigation for students within a college campus.

## Status
